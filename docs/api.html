Maxshot API Documentation

This prototype exposes the intended Maxshot API surface. Production credentials, billing, model routing, and persistence are defined in the product PRD.

Base URL

https://api.maxshot.ai/v1

Authentication

Authorization: Bearer <MAXSHOT_API_KEY>

Chat Completions Example

curl https://api.maxshot.ai/v1/chat/completions \
  -H "Authorization: Bearer $MAXSHOT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "maxshot/auto",
    "messages": [
      { "role": "user", "content": "Explain AIT billing in one paragraph." }
    ],
    "stream": true
  }'

Prototype Scope

  • API keys shown in the prototype are mock credentials.
  • Usage records are local demonstration data.
  • Production billing will meter requests and deduct prepaid credits or AIT.
